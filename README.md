# 🎵 MelodyMath — תרגול מתמטיקה מבוסס-קצב

אפליקציית web לתרגול מתמטיקה יסודית דרך **קצב ומוזיקה** — עברית, מותאמת תוכנית לימודים, בלי התקנה ובלי תלויות (Web Audio API).

## ה-wedge
לא "עוד אפליקציית שיר לוח-כפל". המיקוד: **תרגול מבוסס-קצב לחיזוק ריכוז וזיכרון-עבודה** — הזווית שיש בה ראיה מחקרית (rhythmic-training interventions ל-ADHD/דיסקלקוליה, Frontiers 2019). מיועד לתלמידים שמתקשים בשינון יבש, ולמחנכות שילוב.

## 🎧 שומעים פונקציה (חדש — חטיבה ותיכון)
`functions.html` — גרפים של פונקציות **באוזניים**: סריקת x מנגנת את f(x) כגובה צליל (מיפוי לוגריתמי לסמיטונים, C3–C6), שורשים = נקישות, אסימפטוטת 1/x = בריחת-צליל, ו**סינוס = טון אמיתי** (y=sin(2πft) הוא ממש הצליל — לא מטאפורה). 7 משפחות (לינארית→סינוס), סליידרים שנשמעים חי, ומשחק "נחש את הפונקציה באוזן" עם ניקוד. הבסיס המחקרי החזק: כיוון גובה-צליל ↔ כיוון שיפוע ממופה אוטומטית אצל כולם; וזה הכלי היחיד בעברית לגרפים-בשמיעה (נגישות ללקויי ראייה — חינם לתמיד).

## מה יש היום (חי)
- **לוח הכפל + חיבור** — כל מספר = צליל בסולם; הכפולות מוארות בקצב (Web Audio, אפס תלויות).
- **תרגול בקצב 🥁** — מטרונום מלווה, הקושי (הטמפו) אדפטיבי: מאיץ בתשובה נכונה, מאט בטעות — מאמן inhibitory control.
- **תרגול אדפטיבי** — תרגילים שהתלמיד טעה בהם חוזרים לעיתים תכופות יותר (spaced-repetition-lite).
- **דוח למורה / הורה 📊** — אחוז הצלחה לפי נושא, דגל "שולט / במעקב / צריך עזרה", ונקודות לחיזוק. נשמר מקומית (localStorage), בלי backend.
- משוב מוזיקלי, רצף וכוכבים.

> **סטטוס פדגוגי:** ההנחה ש"מספר = צליל" מסייע לשינון עדיין דורשת אימות בפיילוט מול baseline. הטענה המחקרית הבשלה היא ה-**קצב** (ADHD/דיסקלקוליה), ולכן זה ה-wedge.

## מודל עסקי
web חינמי = top-of-funnel/אמינות. הכנסה מרישוי **B2B/B2G כיתתי-בית-ספרי דרך גפ"ן** (תקציב מוסדי חוזר, ~100% שולי רווח בזכות zero-dep). B2C משני בלבד — מנוי שנתי מראש, לא חודשי.
