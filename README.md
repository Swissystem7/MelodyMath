# 🎵 MelodyMath — לומדים דרך מוזיקה

מערכת ללימוד ילדים דרך **מוזיקה**, לכל הגילאים, לפי תוכנית הלימודים בישראל.
כל מושג נלמד כצליל/מנגינה — הזיכרון המוזיקלי חזק מהזיכרון היבש.

## ה-MVP (חי כאן)
**לוח הכפל דרך מוזיקה** — כל מספר בלוח הוא צליל בסולם. הילד:
1. שומע את "השיר" של הלוח (הכפולות מוארות בקצב),
2. משחק תרגול עם משוב מוזיקלי (תשובה נכונה = אקורד שמח, טעות = צליל עדין),
3. אוסף רצף וכוכבים.

בלי התקנה, בלי תלויות, עובד בדפדפן (Web Audio API).

## החזון
לוח הכפל הוא רק ההתחלה. הרעיון: כל נושא בתוכנית הלימודים (חשבון, שברים, לוח הכפל, יסודות)
נלמד דרך מנגינה מותאמת-גיל — מגן חובה ועד יסודי.

## הפעלה מקומית
פותחים את `index.html` בדפדפן. זהו.
